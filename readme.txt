Aquí se presenta el juego de la pokedex, en el que consiste guardar, evolucionar, avistar y entre otras funciones con la pokedex y sus pokemones.

Se debe utilizar esta linea de compilacion en la terminal en el lugar donde se encuentra el directorio de los archivos
compilar: gcc *.c -Wall -Werror -Wconversion -std=c99 -o pokedex
Y esta linea para poder utilizar el programa para registrar el uso de memoria:
valgrind: valgrind --leak-check=full --track-origins=yes --show-reachable=yes ./pokedex
